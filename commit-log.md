# Commit Activity Log

Auto-generated every 6 hours by the [Commit Activity Tracker](.github/workflows/commit-tracker.yml) workflow.

Each entry shows commits pushed to each repository in the `peak-flow` org during the preceding 6-hour window.

> **Tip:** To track repos outside `peak-flow`, or private repos across the org, add a fine-grained PAT with `Contents: Read` and `Metadata: Read` scopes as a repository secret named `ORG_TOKEN`, then replace `GH_TOKEN` in the workflow with `ORG_TOKEN`.
