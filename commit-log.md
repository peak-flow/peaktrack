# Commit Activity Log

Auto-updated every 6 hours by GitHub Actions (`.github/workflows/commit-log.yml`).
Covers all `peak-flow` repos active in the last 30 days.

> **Setup required:** Add a GitHub Personal Access Token with `repo` scope as a
> repository secret named `GH_PAT` in `peaktrack` → Settings → Secrets and variables → Actions.
> Without it the workflow falls back to `GITHUB_TOKEN`, which only covers public repos.
